# useR! 2026 Conference, Warsaw, Poland

This repository contains the slides for the presentation
*automatedRecLin: Record Linkage Based on an Entropy-Maximizing Classifier*
by Adam Struzik and Maciej Beręsewicz.
The rendered presentation is available in the `useR-2026.html` file.
To build the document locally, render the `useR-2026.qmd` file.
